# 2D-project
VISUAL ASSETS NOT MINE, TAKEN FROM UNITY ASSET STORE AND SPRITERS RESOURCE.

Basic 2D platformer consisting of only 1 level (Has a Day and Night phase), Can easily add more levels without rewriting code though
Movement is based on acceleration (not raw velocity, more physics based) and its easily editable with a few variables to change deceleration, acceleration, max speed, etc
Basic gameplay loop consists of collecting resources at daytime (with a timer based on the level) to prepare for a fight in nighttime 
(Turn based, very basic version of it) player resources save between levels (the 1 level present is loopable, so you can see this ingame)

Concepts for additions include:
A planning phase (general map of the level with approximate item locations) before daytime allowing the player to plan out which items 
he needs the most and what route to take, This would mesh well with the "survival" style i was thinking for the game as you deplete your resources you would be forced to take harder routes for items you need or going for easier ones but risking not having enough time to take everything.

More item active item types
There is only a (jank) double jump potion in the game, this could easily be expanded to add more items usable in daytime/exploration
or nighttime/combat. An obvious example would be a healing item since currently there is only armor (which increases your max health
without healing you)
